Jammming React App

Description

My very first, full fledged react app! This app fetches from Spotify's development API to allow you to search for songs, add them to a custom playlist, and save it with whatever name you choose to your actual spotify account!
I am fairly proud of this one, as I used the power of Axios.js and React.js to create a single page web application that works fairly well. The design aspect of it is simple but nice in my opinion, and I think it fits the purpose of the site well.
As far as I'm aware, the site is safe to log into and will only temporarily hold an authentication token to allow you to create and add a playlist to your account, so feel free to try it By Clicking Here!
