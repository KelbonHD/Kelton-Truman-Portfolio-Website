Description

Primordial Express is a fake restaurant that I made a website for!
I'm pretty proud of this one, as it took me quite a while to research all the different foods that are present on the website, as well as some javascript features I wanted to implement.

But now I was able to make exactly what I wanted, and the increase in my web dev skills are clearly shown with this one. The navigation and other smaller things are a little rough, but I think the page turned out great.
